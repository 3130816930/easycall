📱 EasyCall App 预览

一个为长辈设计的极简视频通话 + 远程协助应用 · 以下为各界面展示

📹

一键视频通话

点击好友即拨通，长辈端自动接听，无需任何操作。WebRTC 高清 P2P 视频。

🟢

自动在线状态

手机开机自动连接服务器，好友列表实时显示在线/离线/通话中状态。

🖥️

远程协助

查看长辈手机屏幕，模拟触摸、打开应用、播放视频，帮长辈操作手机。

🔲

长辈模式

超大按钮极简界面，只有"在线好友 → 点击 → 通话"三步，电话簿都不用管。

🔄 用户流程

📱 爷爷（长辈模式）

  开机 ──→ App 自动连接服务器 ──→ 好友列表显示在线家人
                ↓
       点击家人大按钮 ──→ 自动拨通视频
                ↓
       对方挂断 ──→ 回到主页

──────────────────────────────────────────────────────

📱 家人（普通模式）

  打开 App ──→ 看到好友在线状态 (🟢在线/🟠通话中/⚫离线)
                ↓
        点击好友卡片 ──→ 发起视频通话
         或
        点击 📱 按钮 ──→ 进入远程协助
                ↓
       远程协助:
         · 查看爷爷手机屏幕
         · 点击下方按钮控制:
           ▸ 系统按键（返回/主页/最近）
           ▸ 模拟操作（点击/滑动）
           ▸ 播放视频/打开应用
           ▸ 调节音量

⚙️ 技术栈

backend/ (Node.js)
├── server.js      ← Socket.IO + Express 服务器
├── config.js      ← STUN/TURN 配置
└── package.json   ← 依赖: socket.io, express, sql.js, bcryptjs, jsonwebtoken

app/ (Flutter)
├── lib/
│   ├── main.dart                    ← 应用入口
│   ├── models/user_model.dart       ← 数据模型
│   ├── services/
│   │   ├── api_service.dart         ← REST API (登录/注册/好友管理)
│   │   ├── websocket_service.dart   ← 实时通信 (状态/信令/远程命令)
│   │   ├── webrtc_service.dart      ← 视频通话 + 屏幕共享
│   │   └── remote_control_service.dart ← 远程控制命令定义
│   └── screens/
│       ├── login_screen.dart        ← 登录/注册
│       ├── home_screen.dart         ← 好友列表/长辈模式主页
│       ├── call_screen.dart         ← 视频通话界面
│       └── remote_control_screen.dart ← 远程协助界面
└── android/...   ← Android 平台配置 + AccessibilityService
